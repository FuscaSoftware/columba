Java Security Component Framework.
$Id: package.html,v 1.1 2004/02/16 16:50:29 waffel Exp $ In this package you can found the Java Security Component Framework implementation. The implementation is a hull for other real implementations like gpg command line implementation or something else. The JSCF works like a generic JDBC driver framework where no real JDBC driver is given. The real JDBC drivers are provided from the database developers. The same is for JSCF. This is only a framework and the real implementation for security operations should be provided from the tool developers.

But of course. You can find a small implementation for the gpg command line interface.
